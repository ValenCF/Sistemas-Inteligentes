## Práctica 1 - Sistema Inteligentes

Este repositorio presenta la solución al problema presentado en la asignatura de Sistemas Inteligentes, se pide crear un ascensor de un edificio que funciona en base a percepciones dónde en cada piso hay dos botones uno para subir y el otro para bajar.

El ascensor debe de saber decidir correctamente como subir o bajar eficientemente.
